# Asset Requirements

This folder needs the following icon files:

## Required
- `beeper.png` - Main Beeper icon (512x512 recommended)
  - Used as the extension icon in Raycast

## Optional (for service-specific icons)
If you want custom icons for each service instead of Raycast's built-in icons, add:
- `whatsapp.png`
- `telegram.png`
- `signal.png`
- `instagram.png`
- `messenger.png`
- `discord.png`
- `slack.png`
- `linkedin.png`
- `twitter.png`
- `google-chat.png`
- `google-messages.png`
- `google-voice.png`
- `imessage.png`
- `matrix.png`

## Where to get icons
1. Download Beeper's official icon from their website or app
2. Use a tool like macOS's Preview or Figma to create 512x512 PNGs
3. Service icons can be found at their respective brand resource pages

## Note
The extension will work without custom service icons - it falls back to Raycast's built-in icons with appropriate colors.
