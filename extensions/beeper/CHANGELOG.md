# Beeper Raycast Extension Changelog

## [Initial Version] - 2026-01-15

### Added
- **Open Chat** command to search and open chats from any connected messaging service
- **Search Messages** command to search message history across all services
- **Connected Accounts** command to view all connected messaging services
- Service filter dropdown to filter chats by messaging service
- AI tools for natural language interaction:
  - `open-chat` - Open chats by name
  - `send-message` - Send messages to contacts
  - `list-accounts` - List connected services
- Support for all Beeper-supported messaging services:
  - WhatsApp, Telegram, Signal, Instagram, Messenger
  - Discord, Slack, LinkedIn, X (Twitter)
  - Google Chat, Google Messages, Google Voice
  - iMessage, SMS, Matrix
- Unread message count badges
- Muted chat indicators
- Last message time display
- Connection status checking with helpful error messages
